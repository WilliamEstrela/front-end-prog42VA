Synchronous operation

        
import com.example.myapplication.webservice.cadastrar;
...
SACreprovacao param0 = new SACreprovacao();


SACIncluirReprovacaoServiceSoapBinding service = new SACIncluirReprovacaoServiceSoapBinding("http://localhost:8080/CRUDReprovacaoWS/services/IncluirReprovacaoPort");
SACretorno res = service.inserirReprovacao(param0);
//in res variable you have a value returned from your web service

To invoke a synchronous method, you need to do this in separate thread (for example by using AsyncTask class)
