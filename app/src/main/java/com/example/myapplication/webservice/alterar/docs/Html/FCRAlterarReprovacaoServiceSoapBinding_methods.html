alterarReprovacao

Synchronous operation

        
import com.example.myapplication.webservice.alterar;
...
FCRreprovacao param0 = new FCRreprovacao();


FCRAlterarReprovacaoServiceSoapBinding service = new FCRAlterarReprovacaoServiceSoapBinding("http://localhost:8080/CRUDReprovacaoWS/services/AlterarReprovacaoPort");
Boolean res = service.alterarReprovacao(param0);
//in res variable you have a value returned from your web service

To invoke a synchronous method, you need to do this in separate thread (for example by using AsyncTask class)
